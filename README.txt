COMP 2170 Final Project - Sam Penaflorida

This Project contains a website that lists on the different types of Devil Fruits from the popular show, One piece.
It is a powerscaling way for the characters to gain abilities by eating different Devil Fruits such as Paramecia, Zoan, Logia, 
and a bonus fake Devil Fruit called the SMILE fruit. With Written information on what they are, the types, and their abilities. 
The three main key features are first, the API. Where the API fetches a random Devil Fruit and is displayed with text on the name, 
the type, and it's abilities. The second is the search bar feature at the very bottom of the page, so that once the user is done reading, 
they can search a type of fruit with a keyword in the power, appearing in the same box the user clicks to get a random API Devil Fruit. 
And third, is the navigation menu that opens and closes and guides the user to each section of the website such as the top home section, 
the about section, the types section, and finally, the search section. With the use of DOM manipulation to help with JS logic into the HTML 
content, event listen to make it functional, web storage so the user can keep the latest fruit when refreshed, the public API for the Devil Fruits, 
and a fun and interactive way for the user to learn about One Piece and the powers in Devil fruits.
